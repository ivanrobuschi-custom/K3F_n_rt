# Descrizione del Dispositivo

Questa sezione descrive i modelli disponibili del K3-N RT, il contenuto della confezione, le parti del dispositivo e le segnalazioni di stato.

### Identificazione dei Modelli
Il manuale copre diverse configurazioni del K3-N RT:
- **K3-N STD**: Configurazione base senza display lato cliente.
- **K3-N DSP**: Configurazione con display lato cliente.
- **K3-N WI-FI**: Configurazione con modulo Wi-Fi (accessorio).
- **K3-N BT**: Configurazione con modulo Bluetooth (accessorio).

### Parti del Dispositivo
Le parti principali del dispositivo includono:
- **Parte Superiore e Frontale**: Coperchio vano carta, tasti FEED e OPEN, uscita carta, sigillo fiscale, tasto ON/OFF.
- **Porte di Comunicazione**: Porta di alimentazione, cassetto rendiresto (DK), RS232, USB-B (per PC), USB-A (host), Ethernet, display esterno.
- **Tastiera**: Disponibile in versioni da 30, 35 o 102 tasti (opzionale), con funzioni primarie (Livello 0) e secondarie (Livello 1, attivate dal tasto SHIFT).

### Segnalazioni di Stato (LED)
Il LED di stato fornisce indicazioni visive sul funzionamento del dispositivo:

| Colore | Stato | Descrizione |
| --- | --- | --- |
| Spento | - | Dispositivo spento. |
| BLU | Fisso | Dispositivo acceso, nessun errore. |
| ROSSO | Lampeggiante | Fine carta o apertura vano carta. |
| ROSSO | Fisso | Display esterno non collegato. |
| VIOLA | Lampeggiante | Quasi fine carta, memoria di dettaglio in esaurimento, errore taglierina. |

### Assetti Operativi
Il dispositivo dispone di quattro modalità operative principali, selezionabili con il tasto `Chiave`:
1.  **Cassa Chiusa**: Stato inattivo.
2.  **REGISTRAZIONE**: Per le operazioni di vendita e di cassa.
3.  **LETTURA**: Per la stampa di rapporti gestionali e statistici.
4.  **AZZERAMENTO**: Per l'azzeramento fiscale di fine giornata e azzeramenti periodici.
5.  **PROGRAMMAZIONE**: Per impostare i parametri operativi e le opzioni funzionali.
